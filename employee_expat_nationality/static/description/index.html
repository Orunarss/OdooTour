More Expats in Your Company ?

This module highlights the nationality of employees. This is a useful module for companies which has more expatriates.

Displays nationality in the kanban view of employees with flag.

Displays nationality in the tree view also.

Display employees group by their nationality

Allows to add Residence permit number and it's expiry date.
